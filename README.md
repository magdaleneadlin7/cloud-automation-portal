# Cloud Automation Portal

A web-based cloud automation platform built using Flask, Terraform, and AWS.

## Features

* Provision AWS EC2 instances through a web interface
* Create AWS S3 buckets automatically
* Infrastructure as Code using Terraform
* Automated resource cleanup using configurable timers
* Background execution for improved user experience

## Tech Stack

* Python
* Flask
* Terraform
* AWS EC2
* AWS S3

## Project Workflow

User → Flask Web App → Terraform → AWS Resources
